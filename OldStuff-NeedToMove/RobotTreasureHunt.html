Robot Treasure Hunt

Built using W3.CSS

Algorithms and Python program for the first algorithm below. The input value you enter will give the values for when the treasure is half the distance away you enter. For example if you want to find distance for when the treasure is 1000 units away enter 2000. The first Algorithm is the given one in the example where the robot increases by 5 every time it turns around. The second Algorithm the robot increases its distance by squaring the previous distance traveled starting with 5.
